# Dos命令快速入门

---

**DOS（Disk Operating System / 磁盘操作系统）**

> 单用户、单任务

**进入DOS操作窗口**

> Win + R 进入cmd

---

### 常见的Dos命令

创建目录 md(make director)

```dos
md dir_name
```

进入文件目录 cd

```dos
cd dir_name
```

切换盘符 C:

```dos
#: //#为盘符名
```

显示当前路径所有目录 dir

```dos
dir
```

回退到上一层目录

```dos
cd. //回退两层目录:cd..
```

连续进入目录

```dir
cd dir1/dir2/dir3
```

直接回退到该目录盘符

```dos
cd/ or cd\
```

删除文件目录 rd(remove director)  //如果目录内有内容，则无法使用rd删除

```dos
rd dir_name
```

删除目录下文件 del

```dos
del ###.##
```

删除目录下同一类型(.##)的所有文件

```dos
del *.##
```

清屏

```dos
cls
```
